Элементы
  • element resistor(string node1, string node2, float R)
    присоединить резистор к узлам node1 и node2 с сопротивлением R Ом
  • element capacitor(string node1, string node2, float C)
    присоединить конденсатор к узлам node1 и node2 с ёмкостью C Фарад
  • element inductor(string node1, string node2, float L)
    присоединить катушку индуктивности к узлам node1 и node2 с индуктивностью L Генри
  • element impedance(string node1, string node2, complex Z)
    присоединить комплексное сопротивление к узлам node1 и node2 с импендансом Z
  • element impedance(string node1, string node2, float R, float X)
    присоединить комплексное сопротивление к узлам node1 и node2 с импендансом R + jX
  • element transformer(string node1, string node2, string node3, string node4, float k)
    присоединить идеальный двухобмоточный трансформатор к узлам node1, node2, node3, node4 с коэффициентом трансформации k
  • element autotransformer(string node1, string node2, string node3, float k)
    присоединить идеальный автотрансформатор к узлам node1, node2, node3 с коэффициентом трансформации k
  • element transformer3w(string node1, string node2, string node3, string node4, string node5, string node6, float k1, float k2)
    присоединить идеальный трёхобмоточный трансформатор к узлам node1, node2, node3, node4, node5, node6 с коэффициентами трансформации k1 и k2
  • element line(string node1, string node2)
    присоединить провод с нулевым сопротивлениям к узлам node1 и node2
  • element switch(string node1, string node2, float state)
    присоединить коммутатор к узлам node1 и node2 с состоянием state (0 - выключен, не 0 - включен)
  • element voltageSource(string node1, string node2, complex voltage, float frequency = 0)
    присоединить идеальный источник напряжения к узлам node1 и node2 с комплексным напряжением voltage и частотой frequency Герц
  • element voltageSource(string node1, string node2, float amp, float phase, float frequency = 0)
    присоединить идеальный источник напряжения к узлам node1 и node2 с комплексным напряжением amp·ephase и частотой frequency Герц
  • element currentSource(string node1, string node2, complex current, float frequency = 0)
    присоединить идеальный источник тока к узлам node1 и node2 с комплексным током current и частотой frequency Герц
  • element currentSource(string node1, string node2, float amp, float phase, float frequency = 0)
    присоединить идеальный источник тока к узлам node1 и node2 с комплексным током amp·ephase и частотой frequency Герц
  • void ground(string node)
    присоединить заземление к узлу node
  • element voltage(string node1, string node2)
    добавить в список вывода разницу потенциалов на узлах node2 и node1 → V = φ(node2) - φ(node1)
  • element voltage(element element)
    добавить в список вывода падения напряжения на элементе element
  • element current(element element)
    добавить в список вывода ток на элементе element (положительное направление от node1 к node2)
Вспомогательные функции
  • void print(any arg)
    вывести значение аргумента в текстбоксе вывода
  • float re(complex value)
    получить действительную часть комплексного аргумента value
  • float im(complex value)
    получить мнимую часть комплексного аргумента value
  • float magn(complex value)
    получить амплитуду комплексного аргумента value
  • float phase(complex value)
    получить угол комплексного аргумента value
  • float radians(float value)
    перевести значения аргументов из градусов в радианы
  • float degrees(float value)
    перевести значения аргументов из радиан в градусы
  • complex conj(complex value)
    получить комплексно-сопряжённое значение
